# Skin Disease Detection App

## Overview
A CNN-based deep learning project for skin disease classification using the HAM10000 dataset.

## Features
- Skin Disease Detection
- Severity Analysis
- Treatment Recommendation
- Doctor Referral System
- Monitoring Timeline

## Technologies
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Google Colab

## Dataset
HAM10000 Skin Lesion Dataset
